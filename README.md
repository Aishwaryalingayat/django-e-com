## Instructions

* ### Set-up Instructions
 1. Fork Repository.
 2. Install required packages for ptoject.
 ```bash
 pip install -r requirements.txt
 ```
 3. Setup and activate envoriment.
 4. Run project
 ```bash
 python manage.py migrate
 python manage.py runserver
 ```
